## Projekt używa bibliotek

- axios
- cookies-next
- react-slick
- slick-carousel
- sha.js
- uuid

## Włączenie servera z projektem

```bash
#Pobranie bibliotek
npm run i

#Odpalenie servera developerskiego
npm run dev
```

Następnie należy wejść pod link [http://localhost:3000](http://localhost:3000)

## Budowa projektu

```bash
#Zbudowanie projektu
npm run build

#Dystrybucja
npm run start
```
